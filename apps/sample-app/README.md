# Sample Application (FluxPilot)

This folder contains Kubernetes manifests for a sample application
used to demonstrate GitOps-based deployment in the FluxPilot project.

## Purpose
The sample application shows how application configuration
is stored in Git and automatically synchronized to the cluster.

## Files
- deployment.yaml
- service.yaml
- kustomization.yaml

## Notes
This is a demo application used for learning and testing.
