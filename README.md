# FluxPilot: GitOps-Driven Multi-Cloud Deployment Controller

## Overview
FluxPilot is a GitOps-based project that demonstrates how Kubernetes deployments
can be managed through Git as the single source of truth.

## Folder Structure
- `clusters/` - Cluster GitOps configuration
- `apps/` - Application manifests
- `scripts/` - Helper scripts
- `docs/` - Documentation and notes

## Tech Used
Kubernetes, GitOps, YAML, Shell scripting
